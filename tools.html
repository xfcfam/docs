XF Tools

xftools is the official command-line validator for the CFAM / XF model: it classifies every component into the L×T matrix, runs the 71-rule normative catalog (XF-CFAM-001:2026), and reports a conformance level Λ ∈ {0..4}. It is built as an XF artifact itself — it validates its own source. The documentation below is loaded live from the repository’s README.

Herramientas XF

xftools es el validador oficial de línea de comandos del modelo CFAM / XF: clasifica cada componente en la matriz L×T, ejecuta el catálogo normativo de 71 reglas (XF-CFAM-001:2026) y reporta un nivel de conformidad Λ ∈ {0..4}. Está construido como un artefacto XF en sí mismo — valida su propio código. La documentación de abajo se carga en vivo desde el README del repositorio.
